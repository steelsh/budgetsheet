Объединение файлов

Загрузите файл с формулами (прошлый год) и файл с новыми данными (текущий год).
Приложение автоматически объединит их: формулы из старого файла, входные данные из нового.

значения
→
Результат
Новый лист
Формулы + новые данные,
всё пересчитано
Правила объединения
=
Если в файле 2024 ячейка содержит формулу — она сохраняется. Значение из файла 2025 игнорируется.
↓
Если в файле 2024 ячейка содержит число и в файле 2025 есть значение на том же месте — берётся значение из 2025.
A
Текстовые ячейки (заголовки, метки) берутся из файла 2024 без изменений.
